# Predicción del Consumo de Energía en Costa Rica
Proyecto Final de Programacion 2 desarrollado por Steven Vindas y Claudio Poveda.

Se utilizan los [datos oficiales del ICE](https://apps.grupoice.com/CenceWeb/paginas/CurvaDemanda.html) para consultar la demanda energetica por hora de los ultimos 2 años a nivel nacional y se combina con la temperatura por hora de la provincia de San Jose usando la [API de open meteo](https://open-meteo.com/en/docs/historical-weather-api).

Para ver el dashboard de streamlit ejecute `streamlit run Streamlit.py` o `python -m streamlit run Streamlit.py`
